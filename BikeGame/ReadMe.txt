FOLLOW BELOW STEP TO RUN THE GAME
1. rename RunGame.txt to RunGame.bat 
2.run RunGame.bat

*NOTE:
 PLEASE MAKE SURE THAT JAVA IS INSTALLED IN YOUR PATH AND JAVA PATH IS SET IN 'PATH' ENVIRONMENT VARIABLE.

Instruction for games
use left arrow key to tilt left
use right arrow key to tilt right
for every complete revolution in air player will get 10 point
use up arrow key to move forward
use down arrow key to move back
